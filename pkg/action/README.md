# Actions

An Action is an element of a workflow that is executed within an AlertChain. By providing arguments within a policy, the action can be executed. For more information on how to write a policy, please refer to the [policy documentation](../../docs/policy.md).

## Supported actions

- [github](./github/)
- [jira](./jira/)
- [opsgenie](./opsgenie/)
- [chatgpt](./chatgpt/)
- [slack](./slack/)
- [http](./http)
- [otx](./otx)
- [bigquery](./bigquery)
- [incident.io](./incident_io/)
